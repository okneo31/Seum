## 1. 비전 한 줄

**세움**은 한국어로 작성하는 **시스템 프로그래밍 언어**로, 자체 OS-like 런타임(**나의세상**) 위에서 동작하며, **'담음'** 파일 컨테이너를 데이터 1급 시민으로 가진다. 메모리·타입 안전성에서 **C와 Rust 사이의 새 포지셔닝**.

가까운 선례: Smalltalk-80(언어=환경=이미지) + WebAssembly/WASI(휴대용 런타임) + Inferno/Limbo(언어가 OS) + 한국어 일급 시민.

## 3. Ubiquitous Language (용어 사전)

| 한국어 | 정의 | 통상 영문 | 코드에서 |
|---|---|---|---|
| **나** | 코드 작성자, 1인칭 창조주. 모든 시간·공간의 원점 | self / creator | 암묵적 컨텍스트 |
| **나의세상** | 실행 인스턴스 = OS-like 런타임 단위. 1프로그램 = 1나의세상 | runtime instance | `나의세상 { ... }` |
| **그릇** | 모듈, 도메인 기능 단위 | module / package | `시간`, `웹`, `게임`, `금융`, `담음` |
| **담음** | 파일 컨테이너 형식. `.docx`/`.mp4` 대체 표준. 멀티미디어+문서+데이터 1컨테이너 | container file format | `.담음` 확장자 (가안) |
| **가져오기(X)** | 그릇 X를 나의세상에 로드 | import | `가져오기(시간).` |
| **지금** | 현재 시각 (SST 기준) | now (in SST) | `변수 t = 지금.` |
| **세움 표준시 (SST)** | 1945-08-15 06:00 KST = epoch 0 인 시간 체계 | Seum Standard Time | `지금`이 반환하는 값의 단위 |
| **세움력** | SST 기반 달력 (1945-08-15 = 0년 1월 1일) | Seum Calendar | (확장 그릇에서) |
| **변수** | 가변 바인딩 | mutable binding | `변수 X = ....` |
| **함수** | 호출 가능한 절차 | function | `함수 이름(매개) ... -> 결과 { ... }` |
| **계약** | 자산을 다루는 스마트 컨트랙트 단위 | contract | `계약 이름 { ... }` |
| **자산** | 1급 자원 타입 (BTC, KRW 등) | first-class asset type | `자산 금고: BTC` |
| **위험 { }** | 저수준 / 안전 검사 외 블록 (Rust `unsafe` 대응) | unsafe block | `위험 { 포인터(...) }` |
| **포인터** | 직접 메모리 주소 | pointer | `위험` 블록 내부에서만 |

## 4. MVP 슬라이스 v0.0 — "가장 작은 세움 프로그램이 실행된다"

```세움
가져오기(시간).
보여주기(지금).
```

- **입력**: `.세움` 또는 `.seum` 파일 1개
- **출력**: SST 시각 1줄 (예: `세움력 80년 5월 18일 14:23:01 SST`)
- **실행**: `세움 실행 hello.세움`

---

## 5. v0.0 DO (들어감)

1. **렉서** — Unicode 한글 식별자 + 키워드 + `.` 종결자 + 문자열 리터럴 + 숫자 리터럴
2. **파서** — `가져오기(X)`, `변수 X = E`, `보여주기(E)`, 함수 호출
3. **트리워킹 인터프리터** — AST 직접 평가
4. **그릇 시스템 골격** — `가져오기(X)` 가 표준 라이브러리에서 X 로드
5. **`시간` 그릇** — `지금`, UTC↔SST 변환, 세움력 포맷
6. **최소 `나의세상` 런타임** — 단일 프로세스, stdout, 종료
7. **표준 라이브러리 인터페이스를 freestanding-compatible 모양으로** — 베어메탈 가능성의 씨앗

## 7. Acceptance Criteria (v0.0)

1. ✅ `가져오기(시간). 보여주기(지금).` → SST 시각 출력
2. ✅ `변수 안녕 = "세움". 보여주기(안녕).` → `세움` 출력
3. ✅ `가져오기(시간)` 없이 `지금` 사용 시 명확한 한국어 에러 메시지
4. ✅ 위 셋이 모두 자동화 테스트로 통과
