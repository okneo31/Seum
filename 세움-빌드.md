# 세움 v0.0 — 빌드 / 실행 안내

## 전제 조건

- C++17 컴파일러 (MSVC 2019+ / GCC 9+ / Clang 10+)
- CMake 3.16+
- Git (Catch2 fetch 용)

## 빌드 (Windows / cmd) — **권장**

프로젝트 루트의 래퍼 사용. 새 cmd 창마다 `chcp 65001` 하지 않아도 됨.

```cmd
build                            :: 1회 configure + incremental 빌드
test                             :: 자동 빌드 + 전체 ctest
run examples\hello.세움          :: 예제 실행
```

## 빌드 (Linux/macOS)

```bash
cmake -B build
cmake --build build --parallel
ctest --test-dir build --output-on-failure
./build/seum examples/hello.세움
```

## 수동 빌드 (Windows, 래퍼 안 쓰면)

새 cmd 창마다 다음 한 번 실행 필수 (한글 출력/경로 처리):
```cmd
chcp 65001
```

이후 표준 CMake 흐름:
```cmd
cmake -B build
cmake --build build --config Release
ctest --test-dir build -C Release --output-on-failure
build\Release\seum.exe examples\hello.세움
```

## 기대 출력

```
세움력 81년 5월 18일 11:??:?? SST
```

(년/월/일/시각은 실행 시점에 따라 다름)

## v0.0 슬라이스 확인

|---|---|
| `보여주기("세움").` | `세움` |
| `변수 안녕 = "세움". 보여주기(안녕).` | `세움` |
| `가져오기(시간). 보여주기(지금).` | `세움력 NN년 M월 D일 HH:MM:SS SST` |
