# 세움 (Seum)

> **한국어로 컴퓨터를 움직이는 프로그래밍 언어**

세움은 **한국어로 코드를 쓰는** 프로그래밍 언어입니다.
`print`, `if`, `for` 같은 영어 단어 대신 `보여주기`, `만약`, `반복`을 씁니다.
변수 이름도, 함수 이름도, 오류 메시지도 — 전부 한국어입니다.

```
가져오기(시간).
보여주기(지금).
```

위 두 줄이 실제로 동작하는 세움 프로그램입니다. 실행하면 이렇게 나옵니다:

```
세움력 81년 5월 23일 00:56:23 SST
```

## 1. 프로그래밍 언어가 뭔가요? — 가장 쉬운 설명

컴퓨터는 똑똑해 보이지만, 사람의 말을 그대로 알아듣지는 못합니다.
컴퓨터에게 무언가를 시키려면 **정해진 방식으로 또박또박 적어 줘야** 합니다.
그 "정해진 방식의 글쓰기"가 바로 **프로그래밍 언어**입니다.

문제는, 세상의 거의 모든 프로그래밍 언어가 **영어를 바탕으로** 만들어졌다는 점입니다.
`print`(출력), `if`(만약), `return`(돌려주기), `function`(함수) — 코딩을 처음 배우는 사람은
**프로그래밍 개념 + 영어**를 동시에 넘어야 합니다.

**세움은 그 영어 장벽을 없앱니다.** 한국어를 쓰는 사람이라면, 영어 단어를 외우지 않고도
컴퓨터에게 일을 시킬 수 있습니다.

## 2. 세움은 무엇이 다른가요?

같은 일(두 수를 더해 보여주기)을 하는 코드를 비교해 보세요.

**파이썬 (영어 기반 언어):**
```python
def add(a, b):
    return a + b
print(add(3, 5))
```

**세움 (한국어 기반 언어):**
```
함수 더하기(a, b) -> 결과 {
    돌려주기 a + b.
}
보여주기(더하기(3, 5)).
```

`def` → `함수`, `return` → `돌려주기`, `print` → `보여주기`.
**영어 단어가 한 개도 없습니다.** 코드를 소리 내어 읽으면 그대로 한국어 문장이 됩니다.

> 세움은 단순히 "영어 단어를 한국어로 바꾼 것"이 아닙니다.
> 한국어의 어순과 조사(`~으로`, `~이면`)까지 살려서 설계한 **새로운 언어**입니다.

## 3. 세움의 네 가지 정체성

세움은 네 가지 성격을 모두 가집니다. 어느 하나만 빠져도 세움이 아닙니다.

| 정체성 | 쉬운 설명 |
|---|---|
| **① 한국어가 1등 시민** | 키워드·이름·오류 메시지가 전부 한국어. `(ㄱ:10, ㅅ:20)` 처럼 한글 자모로 좌표도 씁니다. |
| **② 진짜 시스템 언어** | 단순한 학습용 장난감이 아니라, 컴퓨터의 깊은 곳(메모리·하드웨어)까지 다룰 수 있는 "진짜 도구"를 지향합니다. C와 Rust 사이쯤의 위치입니다. |
| **③ 자체 실행 환경 (나의세상)** | 세움 프로그램이 도는 자기만의 세계가 있습니다. 먼 미래에는 운영체제(OS) 수준까지 바라봅니다. |
| **④ '담음' 파일 형식** | 코드·데이터·그림·소리를 **한 파일에 같이** 담는 새 파일 형식(`.담음`). 워드의 `.docx`, 영상의 `.mp4`에 대응하는 세움만의 표준입니다. |

한 줄 요약: **"한국어로 컴퓨터의 모든 것을 다시 설계한다."**

## 4. 세움 코드 맛보기 — 한 줄씩 설명

### 가장 짧은 프로그램

```
보여주기("안녕 세움").
```
- `보여주기(...)` — 괄호 안의 것을 화면에 보여줍니다.
- `"안녕 세움"` — 큰따옴표로 감싼 글자.
- `.` (마침표) — 한 문장의 끝. 한국어 문장처럼 마침표로 끝냅니다.

### 변수 — 값에 이름 붙이기

```
변수 안녕 = "세움".
보여주기(안녕).
```
- `변수 안녕 = "세움"` — `"세움"`이라는 값에 `안녕`이라는 이름을 붙입니다.
- `보여주기(안녕)` — 이제 `안녕`이라고 쓰면 `"세움"`이 나옵니다.

### 만약 — 상황에 따라 다르게

```
만약 (3 > 2) 이면 {
    보여주기("크다").
} 아니면 {
    보여주기("작다").
}
```
- `만약 (조건) 이면 { ... }` — 조건이 맞으면 중괄호 안을 실행.
- `아니면 { ... }` — 조건이 틀리면 이쪽을 실행.

### 반복 — 같은 일을 여러 번

```
반복 3번 {
    보여주기("야!").
}
```
"야!"를 세 번 보여줍니다.

### 시간 — 세움만의 달력 'SST'

```
가져오기(시간).
보여주기(지금).
```
세움에는 자체 달력 **세움 표준시(SST)**가 있습니다.
시작점(원년)은 **1945년 8월 15일, 독도의 첫 일출**입니다.
그래서 2026년은 "세움력 81년"으로 표시됩니다.

### 레코드 — 여러 값을 한 묶음으로 (v0.4 신기능)

```
변수 용사 = (위치: (ㄱ: 10, ㅅ: 20)).
보여주기(용사.위치.ㄱ).
```
- `(ㄱ: 10, ㅅ: 20)` — `ㄱ`(가로) 10, `ㅅ`(세로) 20인 좌표.
- `용사.위치.ㄱ` — 용사의 위치의 가로값 → `10`이 나옵니다.

> 더 많은 예제는 [`examples/`](examples/) 폴더에 있습니다.

## 6. 직접 해보기 — 설치와 실행

세움은 현재 **Windows**에서 동작합니다.

### 준비물 (한 번만)

1. **Windows 10 또는 11**
2. **Visual Studio Build Tools** (마이크로소프트가 무료로 제공)
   - 설치할 때 **"C++ 빌드 도구(C++ build tools)"** 항목을 선택하세요.
   - 이 안에 컴파일러와 CMake가 함께 들어 있습니다.

### 빌드하기 (코드 → 실행 가능한 프로그램으로)

세움 폴더 안에서 `build.bat`를 실행합니다.

```
build.bat
```

처음 한 번은 시간이 조금 걸립니다. `[seum] OK.`가 나오면 성공입니다.

### 실행하기

```
run.bat examples\hello.세움
```

화면에 `세움력 81년 ...` 같은 현재 시각이 나오면 세움이 정상 동작하는 것입니다.

### 테스트 돌려보기 (제대로 만들어졌는지 확인)

```
test.bat
```

`100% tests passed`가 나오면 모든 기능이 정상입니다.

> `build.bat` / `run.bat` / `test.bat`는 복잡한 설정을 자동으로 처리해 주는 도우미입니다.
