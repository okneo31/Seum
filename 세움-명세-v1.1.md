## 3. v0.1 슬라이스 변경 요약

### 추가된 코드
- `third_party/catch2/{catch_amalgamated.{hpp,cpp},LICENSE.txt}` — vendored Catch2
- `build.bat`, `test.bat`, `run.bat` — Windows 래퍼

### 변경된 인터페이스
- `src/value.h`: `is_int`, `is_string`, `is_time`, `is_callable`, `is_getter`, `as_int`, `as_string`, `as_time`, `as_callable`, `as_getter` 추가
- `src/modules/builtin.h`: `OutputSink` 타입 + `register_builtin(env, sink=default_stdout_sink())` 시그니처 변경

### 제거된 코드
- `tests/test_interpreter.cc::capture_run` 의 `freopen` 호출 (DI sink로 교체)
- `src/modules/time.cc` 의 `_mkgmtime` 동적 계산 (상수 하드코드)
- `src/{environment,interpreter}.cc` 의 `std::get_if` / `std::holds_alternative` 직접 호출

### 회귀 검증
- 자동 테스트 32 cases / 61+ assertions / 100% PASS
- 첫 acceptance 4건 모두 회귀 없음

## 4. v0.1 시점 알려진 한계 (v0.2+ 백로그)

- **E. `wmain` UTF-16 → UTF-8**: Windows에서 `세움 실행` 명령행 한글 인자가 ANSI 코드페이지 매개. 현재 CP949에서도 동작하나, 비표준 시스템 로캘에서 fragile 가능. v0.2+ 정리.
- **AST variant 노출**: `parser.h` 의 `Expr` / `Stmt` 변형 타입이 외부 노출. 테스트에서 `std::holds_alternative<ImportStmt>` 직접 사용. 작고 닫힌 집합이므로 v0.0/v0.1에서는 허용.
- **'담음' 포맷 정의**: v0.3 슬라이스에서 진입. 현재까지 미정.
